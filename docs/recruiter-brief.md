# Recruiter Brief

## Project Summary

Built the blueprint and automation content for a small Active Directory security lab. The repo defines a domain layout, supporting Windows hosts, hardening scripts, Sysmon/Wazuh telemetry, Sigma detections, safe simulations, and an incident response runbook.

## What It Shows

- Windows domain architecture planning
- Domain Controller, workstation, and file server design
- Organizational Units and security groups
- Least-privilege identity model
- Group Policy security baseline
- Sysmon endpoint telemetry
- Wazuh detection rules
- Sigma detections mapped to common attack behaviors
- Safe attack simulations for password spraying, privileged group changes, and suspicious PowerShell
- Incident response evidence collection and runbook

## Resume Line

Built an Active Directory security lab blueprint with OU/group design, GPO hardening scripts, Sysmon telemetry, Wazuh/Sigma detections, safe identity-attack simulations, and incident response runbooks.

## Evidence And Limits

- Content validation passes for required files and XML parsing.
- The repo includes seven non-empty PowerShell scripts for setup, hardening, telemetry, reporting, and triage.
- The lab has not yet been executed against live Windows VMs in this workspace.
- Do not claim production deployment, live ransomware testing, or real-domain validation from this repo alone.
