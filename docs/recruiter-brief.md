# Recruiter Brief

## Portfolio Summary

Built an Active Directory Enterprise Lab that recreates a small Windows domain and demonstrates identity administration, Group Policy hardening, detection engineering, attack simulation, and incident response.

## What It Shows

- Windows domain architecture and administration
- Domain Controller, workstation, and file server design
- Organizational Units and security groups
- Least-privilege identity model
- Group Policy security baseline
- Sysmon endpoint telemetry
- Wazuh detection rules
- Sigma detections mapped to common attack behaviors
- Safe attack simulations for password spraying, privileged group changes, and suspicious PowerShell
- Incident response evidence collection and runbook

## Resume Line

Built an Active Directory Enterprise Lab with Windows domain architecture, OU/group design, GPO hardening, Sysmon telemetry, Wazuh/Sigma detections, attack simulations, and incident response runbooks for ransomware-style identity attack scenarios.
